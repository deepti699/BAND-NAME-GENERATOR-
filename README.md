 # Band Name Generator 

This is a simple Python project that generates a band name based on user input.

The program asks for:
- Your city name
- Your pet name

Then it creates a band name.

## Example

Enter your city:
Bangalore

Enter your pet name:
Tommy

Your Band Name:
Bangalore Tommy

## How to Run

1. Download the project
2. Open main.py
3. Run the program
